## 📋 Overview

<a href="#dxoPureRaw">DxO PureRAW</a> serves photographers who shoot in challenging light — high ISO, low light, night photography, astrophotography, indoor sports, documentary — and whose post-production workflows need the cleanest possible RAW file as input before editing begins.

The <a href="#dxoPureRaw">high ISO photography use case</a> is PureRAW's clearest value proposition — photographers who regularly shoot at ISO 3200, 6400, or higher find that DeepPRIME XD2S produces usable, commercially deliverable results from files that standard processing makes too noisy for publication or client delivery. Wedding photographers shooting in dim reception venues, sports photographers under mixed artificial light, documentary photographers in challenging available light — all benefit from the ISO performance that PureRAW unlocks from their existing camera bodies. The <a href="#dxoPureRaw">Lightroom integration workflow</a> fits PureRAW into the standard Lightroom photographer workflow without requiring a new application to learn deeply — PureRAW processes the RAW files and returns them to Lightroom as DNG files, and the Lightroom workflow continues as normal with technically better source files.

The <a href="#dxoPureRaw">astrophotography and night photography use case</a> pushes the extremes of what PureRAW's noise reduction can recover — star fields, Milky Way compositions, and long-exposure night photography that produce extreme noise benefit dramatically from DeepPRIME XD2S processing that recovers detail from noise-dominated images. <a href="#dxoPureRaw">Camera body upgrade alternative</a> — PureRAW effectively improves the high-ISO performance of existing camera bodies to a degree that photographers sometimes describe as the equivalent of upgrading to a higher-performance camera body for low-light work, at a fraction of the camera upgrade cost.

<a href="#dxoPureRaw">Support for all major camera brands</a> with calibrated optical profiles covering cameras and lenses from Canon, Nikon, Sony, Fujifilm, Leica, Panasonic, Olympus, and others — the database of camera and lens combinations continues expanding with new calibrations. <a href="#dxoPureRaw">Processing quality presets</a> balance processing quality against processing time — DeepPRIME XD2S at maximum quality for final output, faster processing modes for preview and evaluation.

<a href="#dxoPureRaw">Apple Silicon optimization</a> uses the M-series chip's Neural Engine for AI processing — dramatically accelerating DeepPRIME processing on Apple Silicon Macs compared to Intel processing, making batch processing of large shoots practical in reasonable time. <a href="#dxoPureRaw">Standalone and plugin operation</a> runs PureRAW as a standalone application for direct RAW processing, or as a plugin from Lightroom Classic for integrated workflow processing.

## 🤔 FAQ

<details>
<summary>Does DxO PureRAW work with Capture One?</summary>

DxO PureRAW produces DNG output that Capture One can open. Direct plugin integration may vary — check current DxO documentation for Capture One workflow details.

</details>

<details>
<summary>How long does DeepPRIME XD2S processing take?</summary>

Processing time depends on the Mac's hardware (Intel vs. Apple Silicon), the number of files, and the RAW file size. Apple Silicon with Neural Engine is significantly faster than Intel. Check DxO's current benchmarks for specific timing data.

</details>

<details>
<summary>Does DxO PureRAW alter the original RAW files?</summary>

No. PureRAW creates new processed output files (DNG, JPEG, or TIFF) and does not modify the original RAW files.

</details>

<details>
<summary>What is the pricing for DxO PureRAW?</summary>

DxO PureRAW is a one-time purchase. Check the DxO website for current pricing.

</details>

<details>
<summary>Does DxO PureRAW support tethered shooting?</summary>

DxO PureRAW focuses on RAW file processing rather than tethered capture. Check current DxO documentation for tethered workflow support details.

</details>

<details>
<summary>Is DxO PureRAW worth it if I already have Lightroom's Denoise AI?</summary>

At moderate ISO values the difference is smaller. At ISO 6400 and above the DeepPRIME XD2S advantage in fine detail preservation is significant and visible. For photographers who regularly shoot at high ISO, the quality difference justifies the additional tool.

</details>
